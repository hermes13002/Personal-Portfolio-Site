# Ayoigbala Soares - Developer Portfolio

Hi! I'm Ayoigbala Soares. This is the repository for my modern, responsive, and interactive developer portfolio. It showcases my most recent projects, technical skills, and work experience as a Mobile Application Developer.

You can view the live site here: [https://personal-portfolio-site-5w5s.onrender.com/](https://personal-portfolio-site-5w5s.onrender.com/)


## Tech Stack

- **Framework**: React 18 with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Custom CSS & Tailwind utilities
- **Deployment**: Render

## Features

- **Responsive Design**: Works flawlessly across desktop, tablet, and mobile devices.
- **Project Showcase**: Detailed project cards with technology chips and live/github links.
- **Work Experience Timeline**: Interactive carousel showcasing my professional history.
- **Skills Grid**: Organized display of my technical competencies.
- **Modern UI/UX**: Dark theme with vibrant violet accent colors, smooth transitions, and glassmorphism elements.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hermes13002/Personal-Portfolio-Site
   cd project
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Deployment

This portfolio is automatically deployed using **Render**. Any pushes to the `main` branch trigger a new production build directly on their platform.

To build it manually for production:
```bash
npm run build
```
The built files will be located in the `dist` directory.
